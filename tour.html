How it works

Main Screen


This is where all the action happens. When you fire up the app you will be presented with this screen from which you have a number of options:

  • Press "Book OFF" to book off immediately for up to 1 hour, this simply sends the "[FSI pin] OFF" text.
  • Press "Book ON" to delete bookings for up to 1 hour, simply sends the "[FSI pin] ON" text.
  • If you press one of the code buttons at the bottom of the screen you will send a formatted text with that code for the times highlighted.
  • You can change the FROM and TO times before sending.
  • The default FROM time is always the next 15 minute slot and you can change the default period to book off for on the SETTINGS page.


Settings Screen


When you first install FireOff you will need to edit a couple of settings before you can enjoy super simple availability texts. There are five settings:

  1. Text Number: This is the phone number you send your availability texts to. If you don't already know this number, ask your manager for it.
  2. FSI Pin code: This is how the system identifies you. If you know how to logon to the FSI availability system, then you will know this number.
  3. No of Hours: You can set the default time period between the FROM (next 15 min slot) and TO times in the booking period. You can use part hours here as well.
  4. Sanity Check: With this set to ON, the app will pop up message informing you what you are about to send. It is easier to understand than the more cryptic text message itself.
  5. SMS Check: It is important to check that your phone sends the message and that you receive a confirmation message from the FSI system, this is simply a reminder to do that which you can switch off if you prefer.
